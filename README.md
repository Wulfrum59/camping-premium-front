    # Camping-premium - Front-end

Interface front-end pour le projet **Camping-premium**.

## Prérequis

- Node.js 18+  
- npm ou yarn  

## Installation

1. Cloner le dépôt :

```bash
git clone  https://github.com/Wulfrum59/camping-premium-front.git
cd camping-premium-front/front

2 . Installer les dépendances :

npm install
ou 
yarn install

3 . Lancer le projet

npm start
ou 
yarn start

- Le projet sera accessible sur http://localhost:3000
